# Delta-exchange
Open-source Python quant bot for BTCUSDT on Delta Exchange (GPLv3). Auto tracks charts, options chain (PCR/Greeks), orderbooks &amp; liquidation maps. Features AI news reader (CPI/FOMC), dynamic ATR risk sizing, and hands-free bracket orders (Entry, Stop-Loss, Take-Profit) with reduce-only safety. Built for Windows 64-bit.
